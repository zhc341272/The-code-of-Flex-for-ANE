# Introduce 简介
How to use my ANE for video of iOS.

如何使用我所编写的ANE。

# Version 版本
Apache Flex 4.14.0

Flash Player 16.0

Air 16.0

# Function 功能
通过flex本身的API，调用iOS端的摄像头，拍摄视频，将视频保存到指定的路径下。

1）通过ANE，调用iOS端的播放器，进行已经录制好的视频的播放功能；

2）通过ANE，调用iOS端媒体图库，将系统图库中的视频文件筛选出来，用户点击之后，返回给Flex端视频的路径，已被接下来的使用。
